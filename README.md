# Crime-Against-Women-Analysis-PowerBI-Dashboard

## 📊 Crime Against Women Analysis & Interactive Power BI Dashboard

### 📌 Project Overview
This project features a comprehensive **interactive Power BI dashboard** that analyzes **Crimes Against Women** in India using official **NCRB (National Crime Records Bureau)** data from 2001 to 2022.

The dashboard transforms complex crime statistics into clear, actionable insights through dynamic visualizations, allowing users to explore year-wise trends, state-wise patterns, crime-type distributions, chargesheet rates, and regional comparisons across six zones of India.

### 🎯 Objectives
- Analyze long-term trends in crimes against women over two decades
- Identify high-risk states and Union Territories
- Understand the distribution of major crime types (Rape, Dowry Death, Cruelty, Assault, etc.)
- Evaluate police efficiency through chargesheet and disposal rates
- Provide zone-wise (North, South, East, West, Central, North-East) comparative insights
- Support data-driven decisions for women’s safety initiatives

### 📂 Dataset
- **Source**: National Crime Records Bureau (NCRB), Ministry of Home Affairs, Government of India
- **Time Period**: 2001 – 2022
- **Key Columns**: State/UT, Year, Crime Types (IPC & SLL), Total Cases, Chargesheet Rate, Crime Rate per lakh female population

### 🛠️ Technologies Used
- **Microsoft Power BI Desktop**
- Power Query (ETL)
- DAX (Data Analysis Expressions) for advanced calculations
- Interactive Visualizations (Line, Bar, Donut, Map, KPI Cards, Slicers)

### 📊 Key Features
- Dynamic year-wise trend analysis with YoY growth
- State-wise ranking by total cases and crime rate
- Crime-type wise percentage distribution
- Chargesheet rate and disposal performance metrics
- Regional zone-wise comparison with map visual
- Interactive filters and drill-down capabilities
